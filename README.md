# 🎬 CineMatch - Movie Recommender System

CineMatch is a content-based movie recommender web application built with **Streamlit**, **scikit-learn**, and **Pillow**. It parses and cleans the TMDB 5000 Movies dataset and provides real-time recommendation updates using a beautiful, responsive user interface.

## 🚀 Quick Start

### 1. Preprocess the Data
First, preprocess the raw TMDB dataset:
```bash
python preprocess.py
```
This merges the movie details and credits datasets, applies text normalization, tokenization, stemming (via NLTK), and exports a clean `movies.pkl` file containing the processed features and metadata.

### 2. Run the Web Application
Start the Streamlit local development server:
```bash
streamlit run app.py
```
By default, the server runs on `http://localhost:8502` (or the next available port). Open this URL in your web browser to interact with the application.

## 🎨 Key Features

*   **⚡ On-the-Fly Cosine Similarity:** Computes similar movies instantly upon user selection, ensuring a minimal memory footprint and fast response times.
*   **🖼️ Dynamic Fallback Posters:** If no TMDb API key is provided (or if the API is down), the app uses Pillow to generate a beautiful, custom typographic poster featuring unique gradient backdrops (hashed based on the movie title).
*   **🔌 Official TMDb Integration:** Support for high-res official posters by adding your TMDb API Key in the sidebar.
*   **🎯 Adaptive Layout:** Click on any recommended movie card to immediately pivot recommendations and browse that movie!
*   **🎛️ Settings Control:** Adjust the number of recommendations dynamically via a slider in the sidebar.
